Pregunta Respuesta
Qué son las etiquetas? Es el contenido que se coloca entre las etiquetas HTML, con la finalidad de darle formato correctamente.
¿Todas las etiquetas HTML vienen en pares? No, algunas no necesitan una etiqueta de cierre.
¿Qué es un mapa de imagen? Permite enlazar muchas páginas web distintas, usando una sola imagen
¿Cual es el significado de la palabra HTML? HyperText Markup Language
Cuantas etiquetas existen en la actualidad de HTML? 142
